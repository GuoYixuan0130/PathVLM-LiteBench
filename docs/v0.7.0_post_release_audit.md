## Release Boundary

The v0.7.0 release artifact contains the artifact-only patch-coordinate heatmap workflow:

- coordinate-aware manifest loading
- patch-score grid aggregation
- heatmap PNG export
- score CSV export
- `render-coordinate-heatmap`
- patch-coordinate heatmap config validation
- synthetic patch-coordinate heatmap demo

The current main branch also contains `score-coordinate-heatmap`, added after the v0.7.0 release tag. That command scores coordinate patches against a text prompt with a CLIP-style model, then writes a score CSV and patch-coordinate heatmap. It is a post-release development feature and is not part of the v0.7.0 release artifact.
