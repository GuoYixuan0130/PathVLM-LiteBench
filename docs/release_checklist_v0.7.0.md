# v0.7.0 Release Readiness Checklist

This checklist defines what should be true before presenting PathVLM-LiteBench as a v0.7.0 patch-coordinate heatmap workflow release.

The goal of v0.7.0 is to add a small, reproducible patch-coordinate heatmap workflow while keeping the project patch-level, artifact-driven, and local-first.

## Release Positioning

v0.7.0 positioning includes:

- coordinate-aware patch manifest loading
- patch score aggregation over `x`/`y` coordinate grids
- PNG heatmap export from precomputed patch scores
- coordinate score CSV export
- artifact-only `render-coordinate-heatmap` CLI
- config validation for `patch_coordinate_heatmap`
- a synthetic patch-coordinate heatmap demo
- offline tests that do not require model inference or real pathology data

v0.7.0 positioning excludes:

- whole-slide image parsing or tiling
- tissue detection, stain normalization, or slide pyramid rendering
- model-inference-driven heatmap generation
- clinical diagnostic claims
- bundled real pathology images or whole-slide image files
- committed generated heatmaps, score CSV files, patches, reports, or local datasets
- required CONCH access or Hugging Face token handling

## Validation Checklist

Run locally before tagging:

```bash
python -m pytest tests
python -m pathvlm_litebench.cli version
python -m pathvlm_litebench.cli models
python -m pathvlm_litebench.cli demos
python -m pathvlm_litebench.cli validate-config configs/patch_coordinate_heatmap_demo_config.json
python -m pathvlm_litebench.cli render-coordinate-heatmap --help
python examples/05_patch_coordinate_heatmap_demo.py
```

Optional artifact-only local check:

```bash
python -m pathvlm_litebench.cli render-coordinate-heatmap \
  --manifest outputs/patch_coordinate_heatmap_demo_synthetic/coordinate_manifest.csv \
  --score-csv outputs/patch_coordinate_heatmap_demo_synthetic/scores.csv \
  --align-by image_path \
  --output outputs/patch_coordinate_heatmap_demo_synthetic/cli_heatmap.png
```

The optional check uses synthetic outputs generated by the demo. These generated artifacts should remain uncommitted.
