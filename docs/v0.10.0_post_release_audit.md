## Released Scope

v0.10.0 released the prompt-set layer for prompt-scored patch-coordinate heatmap workflows:

- prompt-set config support
- `validate-config` support for `patch_coordinate_heatmap_prompt_set`
- committed prompt-set example config
- `score-coordinate-heatmap-prompt-set --dry-run`
- model-free and image-loading-free prompt-set dry-run behavior
- model-backed prompt-set scoring over one coordinate manifest
- per-prompt `scores.csv`, `heatmap.png`, and `metadata.json` artifacts
- `prompt_key` metadata for prompt-set runs
- artifact-only prompt-set comparison CSV and Markdown summaries
- prompt-set workflow documentation
- release notes and release checklist for v0.10.0

The release remains patch-level and local-first. It does not add whole-slide image reading, slide tiling, tissue detection, slide pyramid rendering, clinical diagnosis, clinical decision support, required CONCH access, Hugging Face token handling, bundled datasets, bundled model weights, or committed generated artifacts.
