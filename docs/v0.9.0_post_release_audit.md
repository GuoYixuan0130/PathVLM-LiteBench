## Released Scope

v0.9.0 released the reproducibility and comparison layer for prompt-scored patch-coordinate heatmap workflows:

- scoring metadata sidecar output
- `score-coordinate-heatmap --dry-run`
- `metadata_output` config support
- artifact-only `compare-coordinate-heatmap-scores`
- comparison CSV output
- optional comparison Markdown output
- metadata-assisted comparison summaries
- documentation for metadata, dry-run, and artifact-only comparison
- release notes and release checklist for v0.9.0

The release remains patch-level and local-first. It does not add whole-slide image reading, slide tiling, tissue detection, slide pyramid rendering, clinical diagnosis, clinical decision support, required CONCH access, Hugging Face token handling, bundled datasets, bundled model weights, or committed generated artifacts.
