## Local Smoke Checks

The local audit used the repository virtual environment on Windows. The following commands passed locally:

```bash
python -m pathvlm_litebench.cli version
python -m pathvlm_litebench.cli models
python -m pathvlm_litebench.cli demos
python -m pathvlm_litebench.cli validate-config configs/zero_shot_mhist_clip_sample.json
python -m pathvlm_litebench.cli validate-config configs/zero_shot_mhist_plip_sample.json
python -m pathvlm_litebench.cli validate-config configs/zero_shot_prompt_grid_mhist_sample.json
python -m pathvlm_litebench.cli run-zero-shot-grid --config configs/zero_shot_prompt_grid_mhist_sample.json --output-root outputs/zero_shot_prompt_grid_mhist_sample_v060_post_release_audit --dry-run
python -m pathvlm_litebench.cli summarize-report --help
python -m pathvlm_litebench.cli compare-reports --help
python -m pytest tests
```

Observed results:

- `version` reported `PathVLM-LiteBench version 0.6.0`
- `models` listed CLIP, PLIP, and optional CONCH as implemented registry entries
- `demos` listed the MHIST CLIP and PLIP sample configs, config validation commands, and prompt-grid commands
- CLIP and PLIP MHIST sample configs validated as `zero_shot`
- the MHIST prompt-grid sample config validated as `zero_shot_grid` with 3 models, 3 prompt pairs, and 9 runs
- prompt-grid dry-run expanded 9 planned runs under the requested `--output-root` without model inference
- report summary and comparison CLIs exposed their help text
- tests passed with 159 tests

- `dataset/`
- `outputs/`
- `pathvlm_litebench.egg-info/`
- `.venv/`
